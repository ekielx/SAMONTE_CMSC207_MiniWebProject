# CMSC 207 Mini-Project: Build a Personal Web Profile

This mini-project is a stepping stone towards the final project and focuses on applying foundational web development concepts. Students will create a basic personal web profile, following the principles taught in Clyde Hatter’s book. 
The project will reinforce HTML structure, CSS styling, and basic JavaScript interactivity while keeping the scope manageable for mid-semester progress.

## Project Links
- **Live Demo:** https://ekielx.github.io/SAMONTE_CMSC207_MiniWebProject/
- **GitHub Repository:** https://github.com/ekielx/SAMONTE_CMSC207_MiniWebProject

## Features
- **Responsive Layout:** Optimized for mobile, tablet, and desktop views using CSS Media Queries.
- **Interactive Interests Gallery:** A custom-built modal system that displays image details upon clicking.
- **Modern Navigation:** A sticky/responsive header for easy site navigation.
- **Professional Footer:** Includes contact information and social links, re-aligned for mobile users.

## 💻 Tech Stack
- **HTML5:** Semantic structure for better SEO and accessibility.
- **CSS3:** Custom styling, Flexbox for layouts, and Media Queries for responsiveness.
- **JavaScript:** Logic for modal pop-ups and interactive UI components.

## 📱 Mobile Optimization
The site has been tested for:
- Font scaling on small screens.
- Image responsiveness (max-width controls).
- Stacked layouts for navigation and footer elements.
